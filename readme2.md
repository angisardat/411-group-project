Dependencies -> run npm install (for node.js), npm install dotenv, npm install express, and npm install mongoose 
Instructions -> open our project and in the VSCode terminal, run node server.js and our project should use port 3000. Click on the localhost link to open our project.  
Set-Up Steps -> create an .env folder with GEMINI_API_KEY=your_key by going to ai.google.dev, Explore Models, API Keys, Create API Key, and paste it into your .env folder
Commands to Start the System -> node server.js
